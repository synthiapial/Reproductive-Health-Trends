# Reproductive Health Trends Analysis (Python & Tableau)

## Project Overview

This project analyzes Assisted Reproductive Technology (ART) clinical data to identify trends in treatment volume and success rates across different age groups. Using Python for data cleaning and transformation and Tableau for interactive visualization, the project demonstrates an end-to-end analytics workflow—from raw data preparation to dashboard development.

The goal is to transform complex healthcare data into clear, actionable insights that support data-driven decision-making and quality improvement.

## Technologies Used

* **Python**

  * pandas
  * NumPy
  * Matplotlib
* **Tableau**
* CSV Data Processing

## Project Workflow

### Data Preparation

* Imported and explored raw ART clinical data
* Removed unnecessary columns
* Renamed variables to improve readability
* Cleaned percentage values and converted them into numeric format
* Handled missing values
* Extracted latitude and longitude from geographic coordinates
* Standardized age group labels

### Data Analysis

* Calculated average success rates by age group
* Aggregated treatment cycle counts
* Generated summary datasets for visualization
* Performed exploratory data analysis using Python

### Data Visualization

Created an interactive Tableau dashboard displaying:

* ART success rates by age group
* Treatment cycle volume
* Geographic distribution of clinics
* Key reproductive health trends
